Deskripsi projek: Menganalisis risiko credit sebuah perusahaan berdasarkan loan status yang telah ditransformasikan menjadi good_bad status dengan nilai good dan bad
1. Masuk kedalam google colab
2. Mengimport library python yang dibutuhkan
3. Konekan google colab dengan drive
4. Masukkan dataset ke dalam folder drive dan ubah sesuai nama folder
5. Jalankan atau ubah sesuai dengan kebutuhan: jenis sampling, ukuran sampling, sampling target, model yang digunakan, parameter yang digunakan, jenis parameter tuning, dll.
